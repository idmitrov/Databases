 Homework: SQL Introduction
This document defines the homework assignments from the "Databases" Course @ Software University. Please submit as homework a single zip / rar / 7z archive holding the solutions (source code) of all below described problems.

Problem 1.       What is SQL? What is DML? What is DDL? Recite the most important SQL commands.
Perform a research (e.g. in Google or Wikipedia) and provide an information about SQL, data manipulation language, data definition language and the most important SQL commands.

You should submit a text file called “sql-intro.txt” as a part of your homework.
